## 計算量
 - set - $\mathcal{O}(B)$
 - add, chmin, chmax, chclamp - $\mathcal{O}(B \log B)$
 - get - $\mathcal{O}(1)$
 - count, sum - $\mathcal{O}(B + N/B \log B)$
