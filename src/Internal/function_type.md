| 名前                    | 概要                                                                                                           |
| ----------------------- | -------------------------------------------------------------------------------------------------------------- |
| `is_function_object<F>` | メンバー変数 `value` は クラス `F` が関数オブジェクトなら `true_type`, 違えば `false_type`                     |
| `result_type_t<F>`      | `F` のインスタンスの 関数としての 戻り値の型  ( `F` が関数型 でも 関数オブジェクトクラス でもないなら `void` ) |
