### (※) 第二引数 `partner` について
各頂点の（最大マッチングとは限らない）マッチング相手が記録されている `vector<int>` を渡すことができます。マッチングとして矛盾している場合の挙動は未定義です。一度この関数を実行した後、グラフが少しだけ変化した場合の再計算を効率よく行うために使用できます。

## 使用例

```cpp
#include <iostream>
#include "src/Graph/Graph.hpp"
#include "src/Graph/general_matching.hpp"

int main() {
    // 頂点数N, 辺数M
    int N = 6;
    Graph g(N);
    // 辺を追加
    g.add_edge(0, 1);
    g.add_edge(1, 2);
    g.add_edge(2, 3);
    g.add_edge(3, 4);
    g.add_edge(4, 5);

    auto [matching_edges, partner] = general_matching(g);

    std::cout << "Maximum matching size: " << matching_edges.size() << std::endl;
    // -> Maximum matching size: 3

    for (const auto& edge_id : matching_edges) {
        auto [u, v] = g[edge_id];
        std::cout << "Edge: " << u << " - " << v << std::endl;
    }
    // -> Edge: 0 - 1
    // -> Edge: 2 - 3
    // -> Edge: 4 - 5
}
```
